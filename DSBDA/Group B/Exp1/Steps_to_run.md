# Experiment 1: WordCount - Execution Steps

This document provides detailed step-by-step instructions to compile and run the WordCount MapReduce experiment. Run these commands from the `Exp1` folder.

## Pre-requisites
Ensure Hadoop and Java are installed and running.
1. Start all Hadoop daemons if not already running:
   ```bash
   start-all.sh
   ```
2. Verify daemons are running using `jps`. You should see `NameNode`, `DataNode`, `ResourceManager`, `NodeManager`, etc.

## Execution Steps

### 1. Set the Classpath
Set the Hadoop classpath environment variable to compile the code:
```bash
export HADOOP_CLASSPATH=$(hadoop classpath)
```

### 2. Prepare HDFS Directories
Create input directory on HDFS and upload the input text file.
```bash
hdfs dfs -mkdir -p /WordCountTut/Input
hdfs dfs -put input.txt /WordCountTut/Input/
```

### 3. Compile the Java Code
Compile the `WordCount.java` file.
```bash
mkdir -p classes
javac -classpath ${HADOOP_CLASSPATH} -d classes WordCount.java
```

### 4. Create the JAR File
Package the compiled classes into a JAR file.
```bash
jar -cvf wcjar.jar -C classes/ .
```

### 5. Run the MapReduce Job
Execute the MapReduce job using the created JAR.
```bash
hadoop jar wcjar.jar WordCount /WordCountTut/Input /WordCountTut/Output
```

### 6. View the Output
Check the output generated by the MapReduce job.
```bash
hdfs dfs -cat /WordCountTut/Output/part-r-00000
```
