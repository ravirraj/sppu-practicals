# Practical No 2
**Title:** Assignment 2: Setting up Google Cloud SDK

**Objective:**
- To download, install, and configure the Google Cloud SDK to interact with Google Cloud Platform services.

**Steps to Run:**
Step 1: Download and install Python 3 on your system, as it is required for the standard environment.
Step 2: Download the Google Cloud CLI installer (GoogleCloudSDKInstaller.exe) from the official documentation page and run it.
Step 3: Once installed, open the Google Cloud SDK Shell terminal.
Step 4: Run the command `gcloud auth login`. This will open a web browser where you must log in with your Google account credentials to authorize the SDK.
Step 5: Create a new Google Cloud project by running `gcloud projects create <your_project_name>` (e.g., `gcloud projects create eren300`).
Step 6: Set this newly created project as your active working project by running `gcloud config set project <your_project_name>`.

**Conclusion:** We have successfully installed and configured the Google Cloud SDK, authenticated our account, and set up an active working project.
